Single group
------------

.. py:data:: GET /api/v1/groups/:id

Retrieve the info of a certain group.

Request parameters
^^^^^^^^^^^^^^^^^^

* **id** (*int*) - The group's ID.

Example request
^^^^^^^^^^^^^^^

.. code-block:: bash

   curl -i http://example.com/api/v1/groups/1 \
        -H 'If-None-Match: 820727be5a235d089af4ad66ffeae017'
